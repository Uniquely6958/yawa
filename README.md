# WhatsApp Export Analyzer (GitHub Pages)

Simple static webpage that parses a WhatsApp export text file and shows:
- Total messages
- Unique senders
- Messages per sender
- Media message count

Usage:
1. Open `index.html` in a browser (or host on GitHub Pages).
2. Upload or drag-and-drop a WhatsApp `.txt` export.

To publish on GitHub Pages:
1. Create a new repository and push these files to the `main` branch.
2. In the repo settings enable Pages and set the source to the `main` branch root.
